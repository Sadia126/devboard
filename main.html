1. What are the different ways to select an element in the DOM?

- getElementById()
- getElementsByClassName()
- getElementsByTagName()
- querySelector()
- querySelectorAll()

2. What is the difference between innerHTML, innerText, and textContent?

- innerHTML: Returns or sets the HTML content, including tags.
- innerText: Returns or sets visible text, excluding hidden text.
- textContent: Returns or sets all text, including hidden text.

3. What is event delegation in the DOM?

- Event delegation is when a parent element handles events for its child elements.

4. What is event bubbling in the DOM?

- Event bubbling is when an event propagates from the innermost element to the outer elements.

5. How do you create, add, and remove elements using JavaScript?

- Create: Use document.createElement()
- Add: Use appendChild()
- Remove: Use removeChild()
